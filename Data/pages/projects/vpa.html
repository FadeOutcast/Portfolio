Hatch Post Process Material
GITHUB

The post process material is fully customizable, giving you full control on different aspects like the hash texture used, the hash and primary colors, and the hash texture tiling. You will see in the last image that you are also able to use the normal engine colors combined with the hash texture.

What I enjoyed most about this material was figuring out how to create a shadow mask, which is not available by default in unreal, and figuring out how to layer the hatch texture on top of it in an elegant way.

Please note that due to compression the gameplay may not be as clear as in game.

Unreal subtitle to widget plugin
GITHUB

This is a simple plugin that can be easily integrated into any project. It allows you to display unreals built in subtitles through widgets/UMG rather than through the default display at the bottom. This allows for much better customizability like text color, text size, text location on the screen, and changing the font settings in runtime.

The github link above shows how to use it.
