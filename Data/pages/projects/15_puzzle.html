This game was created for the purpose of learning how to code a game from scratch, using Java and Java Swing. So learning about different design patterns, such as event-driven design, how to debug, how to prevent crashes. Overall this project really helped me develop my coding skills and taught me how to manage a decently sized project (including limiting dependencies and proper naming conventions).

This project also marked my first effective solo usage of Trello for project management, it assisted me in keeping track of the requirements I had set to complete between time frames.

The section of this project which I am most proud of is the multiplayer functionality I added. Whilst peer to peer would have been sufficient for this game, I wanted to test my limits so I ended up developing a dedicated server instead. This server was also developed in Java and was uploaded to an AWS EC2 instance.

The full project can be found in the repository linked above.
