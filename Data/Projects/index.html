Exalted Engine (C++)

API-Agnostic graphics engine built from scratch.

The engine currently only supports OpenGL rendering, however, I am in the process of learning vulkan and plan to integrate that in sometime in the future.

Features

  • Layered Architecture
  • Layer Based Event Dispatch System
  • Abstracted Rendering API
  • Scene Graph Based World Editor based on Unity's editor
  • ImGui Docking Branch Integration
  • Spdlog logging integration

Why Build an Engine?


I wanted to deepen my understanding of game architectures and the procedure of developing tool systems.

Dr. Rich Davison ~ "This is excellent work that clearly demonstrates a good understanding of rendering techniques."

Mathex (C++)

A lightweight math library developed as a learning exercise from scratch.

This library is being developed as I further my knowledge in 3D maths, the goal for this project is to have it support SIMD.

Features

  • Vector 2D
  • Vector 3D
  • Vector 4D
  • Google Test unit testing

Future Development

  • Templated Vectors
  • Matrix Wrappers
  • Quaternions
  • SIMD Integration

Why Build a Math Library?


I always wanted to know how math libraries such as glm work under the hood and I felt that the most rewarding way for me to find out would be by building my own.
